# portal-ky-notes

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面。这些页面通常作为各类笔记、工具或参考资料的轻量级前端呈现，不针对任何特定域名或网站，仅作为静态资源的集中存放与版本管理用途。

## 目录结构

```
portal-ky-notes/
├── pages/          # 存放所有独立 HTML 页面文件
├── assets/         # 页面引用的公共资源（CSS、JS、图片等）
├── index.html      # 可选的索引或导航页面
└── README.md       # 本文件
```

- `pages/`：每个子目录或文件对应一个独立的 HTML 页面，命名应体现内容主题。
- `assets/`：存放多个页面可能共享的样式、脚本或媒体文件，避免重复引用。
- `index.html`：如需要，可在此提供指向各页面的简单链接列表。

## 页面归档说明

- 所有页面均为独立的 HTML 文件，可直接在浏览器中打开，无需服务器环境。
- 页面之间无强依赖关系，可单独查看或修改。
- 归档内容可能涉及个人笔记、工具演示、学习资料等，不包含任何商业或推广信息。
- 页面内容以信息呈现为主，不保证长期稳定更新或对外链接的有效性。
